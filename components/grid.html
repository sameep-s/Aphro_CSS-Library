Grid
Grid is used to manage content in the div. Aphro - CSS uses a 12 column grid system that gives it's user more flexibility when it comes to making layouts.
Basic Grid Example
Add .row class to any conainer to make it a flex container and then add <col-${colsize}> to make it take column. (colsize could be anywhere between 1 to 12)
