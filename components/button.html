Button
Buttons are used to register user's input on the click and then perform the necessary action. use .btn class.( you can choose between different styles of buttons. like btn .btn-primary btn .btn-warning btn .btn-danger ) within <button> element to create buttons. 
<button class="btn btn">Default</button>
<button class="btn btn-primary">Primary</button>
<button class="btn btn-warning">Warning</button>
<button class="btn btn-danger">Danger</button>
